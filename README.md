\# Angular 19 Concept Projects



This repository contains multiple \*\*Angular 19 mini-projects\*\*, each demonstrating a specific concept or feature.  

It serves as a learning lab and a showcase of practical Angular implementations.



---
## üéØ Purpose

This repo is designed to:
- Showcase Angular 19 concepts in isolated, easy-to-understand projects.
- Provide a reference for recruiters and learners to see practical implementations.
- Serve as a personal knowledge base for experimenting with Angular features.

## üìå Notes
- Each project is independent and has its own `package.json`.
- `node_modules` and `dist` folders are excluded via `.gitignore`.
- The `API` folder contains a fake backend for HTTP request demos.
